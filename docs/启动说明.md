# 废弃物AI识别指导投放系统 - 启动说明

## 🚀 快速启动

### 推荐启动方式（最稳定）

```bash
# 方式1: 使用简化版主程序（推荐）
python simple_main.py

# 方式2: 使用启动脚本
python start.py
```

### 其他启动方式

```bash
# 调试版本（用于问题诊断）
python debug_main.py

# 原版本（带启动画面，可能有显示问题）
python main.py
```

## 📋 系统测试

```bash
# 运行系统测试
python test_system.py
```

## 🎯 功能验证

### 启动成功的标志
看到以下日志信息表示启动成功：
```
主窗口显示成功
启动事件循环...
摄像头初始化成功: 设备ID=0
```

### 界面功能
- **左侧**：垃圾分类指导卡片（4种分类）
- **右侧**：检测结果显示区域
- **下方**：状态栏显示系统状态
- **菜单栏**：设置和帮助选项

### 基本操作
1. **摄像头检测**：自动启动，实时显示画面
2. **AI识别**：使用模拟检测（显示可回收物示例）
3. **语音控制**：点击"🔊语音指导"按钮开关语音
4. **全屏模式**：按F11切换全屏，ESC退出全屏

## 🔧 问题排除

### 常见问题

#### 1. 程序启动后立即退出
- **原因**：窗口显示在屏幕外
- **解决**：已修复，使用 `simple_main.py` 启动

#### 2. 摄像头无法打开
```
检查摄像头是否被其他程序占用
确认设备ID配置正确（默认为0）
```

#### 3. 模块导入错误
```bash
# 安装依赖
pip install -r requirements.txt
```

#### 4. 语音功能不可用
```bash
# 安装语音库（可选）
pip install pyttsx3 pygame
```

### 调试方法

#### 查看详细日志
```bash
python debug_main.py
```

#### 检查日志文件
```
logs/waste_detection.log
debug.log
```

## 📁 文件说明

### 主程序文件
- `simple_main.py` - **推荐使用**，简化版主程序
- `main.py` - 原版本，带启动画面
- `debug_main.py` - 调试版本
- `start.py` - 启动脚本

### 配置文件
- `config/system_config.json` - 系统配置
- `config/waste_classification.json` - 垃圾分类配置

### 测试文件
- `test_system.py` - 系统测试脚本

## 🎨 界面特色

### 垃圾分类
- **🔵 可回收物**：塑料瓶、纸张、金属等
- **🔴 有害垃圾**：电池、药品、化学品等  
- **🟤 湿垃圾**：食物残渣、果皮、茶叶渣等
- **⚫ 干垃圾**：其他垃圾、污损纸张等

### 视觉效果
- 渐变背景和现代化设计
- 分类卡片动态高亮
- 检测结果实时显示
- 置信度进度条

## 📈 扩展功能

### 添加真实AI模型
1. 将RKNN模型文件放入 `models/` 目录
2. 更新 `config/system_config.json` 中的模型路径
3. 修改 `config/waste_classification.json` 中的类别映射

### 自定义分类
编辑 `config/waste_classification.json` 文件：
- 添加新的垃圾分类
- 修改分类颜色和图标
- 更新投放指导文本

## 💡 使用建议

1. **首次运行**：建议先运行 `test_system.py` 检查系统状态
2. **日常使用**：使用 `python simple_main.py` 启动
3. **问题诊断**：使用 `python debug_main.py` 获取详细信息
4. **功能完整**：安装可选依赖 `pyttsx3` 和 `pygame` 获得完整体验
